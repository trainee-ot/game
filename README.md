# Game

本仓库包含两个独立的网页小应用，均为纯 HTML 文件，**无需安装任何软件**，直接用浏览器打开即可使用。

---

## 📂 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 过马路小游戏 |
| `naxx.html` | 魔兽世界 Naxx 天启四骑士团本战斗辅助工具 |

---

## 🚀 如何打开

### 方法一：直接在本地打开（最简单）

1. 点击本页面右上角的绿色 **Code** 按钮，选择 **Download ZIP**。
2. 解压下载的 ZIP 文件到任意文件夹。
3. 进入解压后的文件夹，**双击** 想要打开的 HTML 文件：
   - `index.html` → 过马路小游戏
   - `naxx.html` → 四骑士战斗辅助
4. 文件会自动在默认浏览器（Chrome / Edge / Firefox 均可）中打开。

> **提示：** 也可以先打开浏览器，然后把 HTML 文件直接拖入浏览器窗口。

### 方法二：通过 GitHub Pages 在线访问

如果仓库已开启 GitHub Pages，可直接访问：

- 过马路小游戏：`https://<用户名>.github.io/<仓库名>/`
- 四骑士辅助工具：`https://<用户名>.github.io/<仓库名>/naxx.html`

开启 GitHub Pages 的步骤：

1. 进入仓库页面，点击顶部 **Settings**。
2. 左侧菜单选择 **Pages**。
3. 在 **Branch** 下拉菜单中选择 `main`，目录选 `/ (root)`，点击 **Save**。
4. 稍等片刻，刷新页面即可看到在线访问链接。

## 🎮 四骑士战斗辅助（naxx.html）使用说明

1. 打开 `naxx.html`。
2. 在配置界面依次输入 **7 位坦克**和 **12 位治疗**的名字。
3. 在下方选择坦克 G（矮子）的专属治疗（从治疗 5/6 和 7/8 各选一人）。
4. 点击 **▶ 开始战斗** 按钮，战斗计时器启动。
5. 界面将每 12 秒自动累加层数，并实时显示每个 Boss 当前负责的坦克和治疗。
6. 需要重新配置时，点击 **■ 停止战斗** 返回配置界面。
